⚙️ Configuração GitHub

Os dados serão salvos no seu repositório GitHub e ficam acessíveis em qualquer computador.

Como criar o token:
GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
Marque o escopo: repo (acesso total ao repositório)
O token começa com ghp_
Os dados são salvos em db/dados.json no repositório.
